# Prueba

Pagina web estatica de ejemplo creada con HTML, CSS y JavaScript.

## Archivos

- `index.html`: estructura principal del sitio.
- `styles.css`: diseno visual, responsive y animaciones.
- `script.js`: menu movil, tema claro/oscuro, contadores, animaciones al hacer scroll y formulario de ejemplo.
- `assets/`: imagenes locales generadas para la pagina.

## Uso local

Abre `index.html` directamente en el navegador.
